# Mindful Growth - Beta Test Version 1.0.0

Thank you for participating in the Mindful Growth beta test!

## Installation
1. Download the APK from the provided link
2. Enable "Install from Unknown Sources" on your device
3. Open the APK and install
4. Grant all requested permissions

## What to Test
- **Stats Screen**: View your screen time over 7 days
- **Customize Screen**: Purchase and select different trees
- **Settings**: Configure Focus Mode preferences
- **Focus Mode**: Test the lock screen overlay
- **Tree Growth**: Verify trees grow during focus sessions

## How to Report Issues
1. Take screenshots of any bugs
2. Note the exact steps to reproduce
3. Include your device model and Android version
4. Submit via: [feedback form link]

## Known Limitations
- Works on Android 13+ only
- Some animations may be slower on older devices
- Usage stats require permission grant

## Privacy Note
- All data stays on your device
- No analytics or tracking
- No internet connection required
